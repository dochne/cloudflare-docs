Background color to add underneath the image. Applies to images with transparency (for example, PNG) and images resized with `fit=pad`. Accepts colors in hex format (e.g. #FF0000), as well as using CSS4 modern syntax such as `rgb(255 255 0)` or `rgba(255 255 0 100)`. 

<Tabs>
  <TabItem label="URL format">
    ```js 
    background=%23RRGGBB

    OR

    background=red
    ```
  </TabItem>
  <TabItem label="Workers">
  ```js 
  cf: {image: {background: "#RRGGBB"}}
  ```
  </TabItem>
</Tabs>
